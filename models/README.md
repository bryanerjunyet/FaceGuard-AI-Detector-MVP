# Models

Store deployed checkpoint files for the detector here.

- Active pretrained checkpoints live in `models/pretrained/`.
- The backend load path is configured in `config/settings.py`.
- Pretrained checkpoint names currently include `vit.pth`, `xception.pth`, and `pg_fdd.pth`.
- Dataset and attribution notes are documented in `models/THIRD_PARTY_NOTICES.md`.
