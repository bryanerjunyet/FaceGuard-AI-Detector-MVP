# Third-Party Notices

This repository contains the FaceGuard detector application and does not redistribute the upstream training dataset.

## AI-Face dataset and benchmark

The FaceGuard model training work is based on the AI-Face dataset and benchmark introduced in:

Li Lin, Santosh, Mingyang Wu, Xin Wang, and Shu Hu.
"AI-Face: A Million-Scale Demographically Annotated AI-Generated Face Dataset and Fairness Benchmark."
Proceedings of the IEEE/CVF Conference on Computer Vision and Pattern Recognition (CVPR), 2025.

Research Paper: https://arxiv.org/abs/2406.00783

Upstream repository: https://github.com/Purdue-M2/AI-Face-FairnessBench

Notes:
- This detector repository does not include the AI-Face dataset itself.
- Access to the dataset, annotations, checkpoints, and any related training assets remains subject to the original authors' distribution terms and license.
- The upstream training README states that the AI-Face dataset is licensed under `CC BY-NC-ND 4.0`. Refer to the original project materials for the authoritative license text and usage conditions.
